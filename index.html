Como alimentar os dados automaticamente

Este site já está pronto para ler um arquivo JSON ou uma API. Você só precisa apontar a origem dos dados.

Opção 1 — JSON local / rede

Ideal para começar rápido. Gere o arquivo dados_coa_cana.json a partir do Excel, Power Automate, Python ou ERP e coloque na mesma pasta do HTML ou em uma pasta compartilhada.

Opção 2 — API corporativa

Substitua a constante DATA_URL por um endpoint como https://seu-servidor/api/coa/cana. O site atualiza automaticamente sem mexer no HTML.

Opção 3 — SharePoint / Power BI / SQL

Você pode publicar os dados em uma Lista do SharePoint, em um arquivo JSON sincronizado no OneDrive, ou expor uma API acima do SQL Server/PostgreSQL.

Dica prática: o caminho mais simples para começar é Excel → Power Automate / Script → JSON, e depois evoluir para API + banco.
